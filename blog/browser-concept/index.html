Reimagining the web browser
16 September 2021

Originally published on 12 August 2021

There have been a few key events that occurred in the browser space last June. Firefox received the Proton interface, Safari has had a major visual overhaul, Opera received its Pinboard feature, and Chrome ended an experiment on simplifying the URL. What intrigued me the most, however, is the new Safari. Its new interface does not conform to the traditional browser layout. Instead, it combines the tab bar, the address bar, and the toolbar into a single row of buttons while at the same time reducing visual clutter. On top of this, the tab bar now matches the color of the page. The result is an immersive and streamlined look that is very easy on the eyes.

Firefox has been my browser of choice since the mid-2000s. Naturally, I wondered, what would it look like if it resembled the new Safari?

Let's attempt to answer this question, and explore new functionalities that could make Firefox a better browser.

While this concept would probably solve some of the problems users have today, it is definitely not the be-all and end-all. Design trends are constantly changing and evolving. After a while, new issues would undoubtedly surface, and that would require new innovative solutions. If this concept were to be implemented, it would need some ironing out and extensive user testing.
